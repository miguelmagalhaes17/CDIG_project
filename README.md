Formato: X_Y_Z\[nome do esquemático\],

Sempre que fizermos uma alteração significativa (tipo adicionar componentes etc), mudamos o número Y
Sempre que dentro do mesmo esquemático alterarmos valores, fizermos vários testes diferentes, alteramos o número Z, que neste caso não significa estar mais avançada ou não

O X deve ficar sempre a 0 até completarmos o projeto todo, aí mudamos para 1. Dentro da versão 1_Y_Z, o Y vai ficar sempre a 0, e os testes vão alterar apenas o Z.
